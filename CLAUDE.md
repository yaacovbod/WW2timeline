# פרויקט: WW2timeline (ארועים בזמן)

## מטרת הפרויקט
משחק לימודי אינטראקטיבי לסידור אירועי מלחמת העולם השנייה והשואה בסדר כרונולוגי. השחקן מקבל 8 אירועים אקראיים ומסדר אותם על ציר הזמן. יש 3 חיים ופנלטי על טעות.

## טכנולוגיות
- Next.js 16 + React 19 + TypeScript
- Tailwind CSS 4
- Vercel Blob (לשמירת לוח התוצאות)
- שם החבילה: `arev-bazman`

## מבנה קבצים מרכזי
- `src/components/Game.tsx` - קומפוננטה ראשית עם כל לוגיקת המשחק
- `src/data/events.ts` - כל האירועים (כ-40 אירועים מ-1933 עד 1945) עם תמונות, תאריכים ותיאורים בעברית
- `src/data/schools_gen.ts` - רשימת בתי הספר
- `src/app/api/scores/route.ts` - API לשמירת וקריאת לוח התוצאות ב-Vercel Blob
- `src/app/admin/page.tsx` - דף אדמין לצפייה ומחיקת ציונים
- `public/pics/` - תמונות האירועים בפורמט JPEG ממוספר

## לוח תוצאות ואדמין
- ציונים נשמרים ב-Vercel Blob לפי שם בית ספר
- דף אדמין מוגן בסיסמה דרך `ADMIN_SECRET` env var
- מחיקת ציון: `DELETE /api/scores?secret=...&index=N`

## קהל יעד
תלמידי בית ספר בישראל הלומדים על השואה ומלחמת העולם השנייה
